# 3. Jetpack WorkManager

### Jetpack WorkManager 이란?
안드로이드에서 '백그라운드 작업'을 가장 확실하게 예약하고 실행하는 기술이다.
'30분마다', '폰이 충전 중일 때' 등 특정 조건에서 코드를 실행시킬 수 있다.
안드로이드 OS가 배터리를 아끼려고 앱을 강제로 재워도, `WorkManager`는 OS를 설득하여 결국엔 실행을 보장한다.

### 본 프로젝트에 필요한 이유
* [Phase 2] 시간의 흐름: 본 프로젝트의 핵심이다. 사용자가 앱을 보지 않을 때에도 펫은 스스로 배가 고파져야 한다.
* `WorkManager`가 1시간마다(혹은 OS가 허용하는 주기로) '심장'처럼 뛰면서 `Worker`를 실행시킨다.
* `Worker`가 실행되면, `DataStore`에서 '마지막 업데이트 시간'을 읽어 와서 '타임스탬프 기반 로직'을 실행하고,
새 상태를 다시 `DataStore`에 저장한다.
* '살아있는 펫을 위한 백그라운드 로직'의 핵심 엔진이다.
